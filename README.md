# SpotKeys
A C++ program that allows simple manipulation of Spotify's webplayer (play, pause, next, previous, volume up, and volume down) with keyboard shortcuts.

## Set Up

- Install dependencies.
- Add path to Geckodriver and Firefox to SpotKeys_Settings.txt as well as path to any desired addon.
- Build

## Dependencies
Python 3.8+
Selenium 3.141+ (Python)
Geckodriver 0.29+
Firefox 86+
